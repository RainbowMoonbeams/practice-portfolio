---- Projects ----

Fortune Teller

In this interactive game, the user is asked to input their name, age, birth month, favorite color, and number of siblings. The fortune is determined by each of the variables and the fortune will change each time you play the game if you input different responses.

  • Concepts used in the App:
  • Accepting User Input
  • Creating Variables & Basic Types
  • Using Operators & Expressions
  • Conditionals
  • Strings


Virtual Pet

Remember Tomagochi and GigaPets? This is a very similar game! You'll interact with a pet named Albus by feeding him, watering him, and playing with him. Albus is self-sufficient only when absolutely necessary, and in the most annoying ways. Play along to see how far you can make it as his companion.

  • Concepts used in the App:
  • Accepting User Input
  • Instance Variables
  • Methods
  • Looping
  • Interface
  • Main Method App

Virtual Pets Run Amok

In this hair-raising game interact with and care for multiple organic and robotic cats and dogs. These pets need lots of attention, including cleaning individual dog cages & the communal cat litter box, taking the robotic and organic dogs for walks, feeding organic pets, maintaining robotic pets by oiling and recharging them. You can also adopt from the shelter, as well as surrender an organic or robotic cat or dog to the shelter.

  • Polymorphism
  • Looping & Tick Method
  • Interface
